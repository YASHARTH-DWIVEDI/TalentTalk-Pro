# TalentTalk-Pro
TalentTalk Pro : An AI-powered Mock interview system. Features text/voice modes (Whisper/AssemblyAI), dynamic resume analysis, and customizable questions. Uses LangGraph to conduct multi-stage interviews, evaluate responses, and generate comprehensive PDF reports, Revolutionizing training with automated, insightful assessments
